# AGENTS.md — basic-workspace (example)

> This is a standalone example workspace, not a thin sub-repo — it exists to
> show what a repo using passdown actually looks like: an inbox note, a
> processed OpenSpec change (`openspec/changes/pkg-0001-demo/`), and a
> session log. See the passdown repo's own `README.md` and
> `templates/AGENTS.thin.md` for what to copy into a real project.

## Repo-specific notes

Fictional CLI tool `demo-cli`. There is no real source code here — this
workspace only exists to exercise the passdown skills end to end.

## passdown

- inbox: docs/inbox/
- planning: openspec (schema `passdown`)
- log_dir: docs/log/
- log_language: en
- executors: main
